# archiver
Python File &amp; Folder Compressor

---

# 압축법

Win + R ==> wt 입력 후 엔터
<br><br>
`python archiver.py c "압축 대상" "결과"`

---

# 압축 해제법

Win + R ==> wt 입력 후 엔터
<br><br>
`python archiver.py d "압축 해제 대상"`
